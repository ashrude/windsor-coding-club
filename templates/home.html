Windsor Coding Club

Are you interested in coding? The Coding Club is a place where you can learn code, or improve your skills at your own pace, while learning from everyone around you.

If you're interested come by for a meeting and see if you like it. We meet Thursdays at lunch in room 205.

About Us

We are the Windsor Coding Club. A place where you can hang out and talk about code. Don't worry if you don't know anything about code yet, most of us only know one or two coding languages and are still a while away from mastering any language. Even this page is designed and developed by people using HTML and CSS for the first time.
